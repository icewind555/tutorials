# 如何参与贡献

* 执行 `npm install` 安装依赖
* 执行 `npm run serve` 启动服务器
* 按照 [规范](./common/styles/markdown-style.md) 修改
* 执行 `npm run lint` 检查代码
* 提交 Pull Request
